常见问题解答（FAQ）

Q: 如何获取并设置 API Key？
1. 访问阿里云通义千问官网
2. 注册/登录阿里云账号
3. 在控制台中创建 API Key
4. 将获取的 API Key 复制到应用的设置页面中
Q: 为什么有些图片无法识别？
请确保：
- 图片清晰可读
- 图片大小不超过10MB
- 网络连接稳定
- API Key 设置正确
Q: 如何设置提醒？
1. 创建待办事项时，系统会自动识别时间信息
2. 如果识别到时间信息，系统会自动设置提醒
3. 您也可以手动修改提醒时间
Q: 数据是否安全？
- 所有待办事项数据均存储在本地设备中
- 图片分析仅在必要时传输到服务器
- 我们不会存储您的任何个人信息
- API Key 使用安全加密存储
